# lfguard 0.8.2

`lfguard 0.8.2` is a patch release for the advisory evidence contract.

## Contract Fixes

- Adds top-level `hard_block` to `review/summary.json`, matching the documented
  service decision contract.
- Adds stable `docs_anchor` fields alongside `docs_url` in lint, audit, plan,
  review blocking reasons, and planned grant evidence.
- Documents `docs_url` as a live documentation link and `docs_anchor` as the
  stable key for stored evidence and service UI mappings.

## Catalog Coverage

- Adds catalog rows for grant drift audit findings:
  `GRANT_MISSING`, `GRANT_PERMISSIONS_MISSING`,
  `GRANT_PERMISSIONS_UNMANAGED`, and `GRANT_UNMANAGED`.
- Strengthens catalog tests so dynamically generated grant audit findings must
  have stable catalog metadata.
