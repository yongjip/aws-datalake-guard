# lfpolicy 0.10.0

`lfpolicy 0.10.0` renames the project from `lfguard` to better match its role as
an advisory Lake Formation policy review and evidence layer. It does not add
AWS write execution or approval orchestration.

## Breaking rename

- Install with `python -m pip install lfpolicy`.
- Run commands with `lfpolicy`.
- Import the public Python API from `lfpolicy`.
- Consume report schema identifiers under `lfpolicy.*.v1`.
- Read `lfpolicy_version` from review manifest and summary evidence.
- Use `.lfpolicy/`, `lfpolicy-*` artifacts, and `LFPOLICY_*` live-test
  environment variables.

The report data shapes and advisory decision semantics remain unchanged.
