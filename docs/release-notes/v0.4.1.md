# lfguard 0.4.1

`lfguard 0.4.1` is a small patch release for live apply of named LF-Tag
expression changes.

## Fixed

- Fixes live apply for `lf_tag_expression.create`,
  `lf_tag_expression.update`, and `lf_tag_expression.delete` when no catalog ID
  is configured. Internal lower-case `catalog_id` payload keys are now removed
  before calling boto3 even when the value is empty.

## Tests

- Adds botocore `Stubber` coverage for named LF-Tag expression create, update,
  and delete request shapes with and without `CatalogId`.

## Upgrade

```bash
python -m pip install --upgrade lfguard
lfguard --version
```
