# lfguard 0.7.2

`lfguard 0.7.2` improves agent discoverability and documentation coverage for
LLM-safe governance integrations.

## Agent Guidance

- Adds `AGENTS.md` as the repository-level coding-agent instruction file.
- Adds `CLAUDE.md` as a thin Claude Code pointer to the shared agent
  instructions.
- Adds `llms.txt` as a compact LLM-readable documentation index.
- Adds `docs/llm-agent-integration.md` with runtime decision rules for DMS
  workflows and service-embedded LLM agents.

## Documentation And Packaging

- Links the LLM agent guide from README, CLI docs, and report-format docs.
- Includes `AGENTS.md`, `CLAUDE.md`, and `llms.txt` in source distributions.
- Adds the LLM integration guide to PyPI project URLs.
- Adds regression tests for agent-document links, manifest inclusion, and
  project URL coverage.
- Ignores macOS `.DS_Store` metadata files.
